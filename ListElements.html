List Elements

Lists are used to group related items together in HTML. There are two main types of lists: ordered lists and unordered lists.

Ordered Lists

An ordered list is a list where the items are numbered. It is created using the <ol> tag. Each item in the list is defined using the <li> tag.

<ol>
                        <li>Item 1</li>
                        <li>Item 2</li>
                        <li>Item 3</li>
                        </ol>

Unordered Lists

An unordered list is a list where the items are not numbered, but are instead marked with bullet points. It is created using the <ul> tag. Each item in the list is also defined using the <li> tag.

<ul>       
                        <li>Item A</li>
                        <li>Item B</li>
                        <li>Item C</li>
                        </ul>

Nested Lists

Lists can also be nested, meaning you can have a list within a list. This is useful for creating sublists.
