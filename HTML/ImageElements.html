Image Elements

Images can be added to HTML documents using the <img> tag.

Image Element Syntax

The basic syntax for an image element is:

<img src="image.jpg" alt="Description" width="300" height="200">

Attributes of the Image Element

  • src: Specifies the path to the image file.
  • alt: Provides alternative text for the image if it cannot be displayed.
  • width: Sets the width of the image in pixels.
  • height: Sets the height of the image in pixels.
  • border: Sets the border around the image (deprecated in HTML5, use CSS instead).
  • vspace: Sets the vertical space around the image (deprecated in HTML5, use CSS instead).
  • hspace: Sets the horizontal space around the image (deprecated in HTML5, use CSS instead).
